# Indicador-de-Bateria-de-aparelhos-eletronicos-
1:
1. O problema indentificado é a falta de bateria, quando esquece de colocar carregar.
2. Esse problema acontece em varios lugares, as pessoas esquecem de por o aparelho carregar e acabam ficando sem carga para o resto do dia.
3. As pessoas afetadas são as que esquecem com frequencia de por o aparelho carregar.
4. Um programa de computador poderia fazer com que as pessoas lembrem de por seu aparelho no carregador.

2:
1. O nome do programa é Indicador de Bateria para aparelhos eletronicos.
2. O objetivo é lembrar as pessoas de carregarem seus aparelhos.
3. O usuario devera digitar a porcentagem que a bateria dele se encontra.
4. Será exibido avisos para lembrarem de por no carregador.
5. A regra de negócio será a exibição de avisos caso necessario.
